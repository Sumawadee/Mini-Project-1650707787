ก่อนใช้งานให้พิมพ์คำสั่งใน terminal หรือ command prompt
- mkdir book-inventory-system
- cd book-inventory-system
- npm init -y
- npm install typescript @types/node --save-dev
- npx tsc --init

วิธีรันโปรแกรม
- npx tsc
- node dist/index.js

ฟังก์ชันของโปรแกรม
- Add Book
- Update Book
- Delete Book
- Search Books
- List All Books
